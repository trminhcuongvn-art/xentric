# Xentric Affiliate

Standalone GitHub Pages repo for Xentric affiliate landing pages.

Target URL after deployment:

- https://trminhcuongvn-art.github.io/xentric/
- https://trminhcuongvn-art.github.io/xentric/tools/reclaim-ai/
- https://trminhcuongvn-art.github.io/xentric/tools/customgpt/

## Deploy steps

1. Create GitHub repo `trminhcuongvn-art/xentric`.
2. Commit and push branch `main`.
3. Enable GitHub Pages: branch `main`, folder `/`.
4. Replace `[AFFILIATE_LINK_PLACEHOLDER]` after affiliate accounts are approved.

## Notes

- GitHub Pages default path uses `/xentric/` base.
- Affiliate disclosure is included on landing pages.
